Starting WebScarab

The installer will create shortcuts to the webscarab.bat file on win32. You can either run WebScarab by selecting the shortcut, or by browsing to the directory in which you installed WebScarab, and running the webscarab.bat file yourself. You can also run WebScarab by double clicking on the webscarab.jar file, if the .jar extension has been associated with the JRE.

On Unix platforms, you can run WebScarab by running the webscarab.sh file.
